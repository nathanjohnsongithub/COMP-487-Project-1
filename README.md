## Project Definition
For our project, we want to test how a model detects real vs AI-generated images of landscapes from 3 different datasets. Specifically, a dataset from 10 years ago, a dataset from 2-3 years ago, and a dataset that was created within the last year. 

## Dataset Link
- Early models | [CIFAKE](https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images). It has 60,000 real images from CIFAR-10 and 60,000 AI-generated images. This are low res images
- Middle tier models | [GenImage](https://github.com/GenImage-Dataset/GenImage) GenImage is a million-scale benchmark with over one million real/fake pairs.
- Modern models | [Synthbuster](https://zenodo.org/records/10066460) Synthbuster includes images from newer generators such as DALL·E 2, DALL·E 3, Adobe Firefly, Midjourney v5, Stable Diffusion 1.3/1.4/2, Stable Diffusion XL, and Glide.
